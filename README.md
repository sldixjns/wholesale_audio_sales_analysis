#  Анализ оптовых продаж | Sales Performance Analytics

##  Цель проекта
Комплексный анализ данных оптовых продаж для выявления точек роста, оптимизации бизнес-процессов и повышения общей эффективности компании.

##  Технологический стек
- **Python** — основной язык анализа
- **Pandas / NumPy** — обработка и трансформация данных
- **Matplotlib / Seaborn** — построение интерактивных визуализаций
- **Jupyter Notebook** — среда для исследовательского анализа

##  Что было сделано

### 1. Подготовка и консолидация данных
- Автоматизированный сбор данных из разрозненных источников (CSV-файлы, организованные по папкам)
- Объединение таблиц: заказы, статусы, товарный каталог, финансовые данные
- Очистка и стандартизация данных, обработка пропусков и аномалий

### 2. Расчёт ключевых бизнес-метрик
- **GMV (общая выручка)** и **AOV (средний чек)** за анализируемый период
- Конверсия по этапам воронки продаж
- Доли выручки по различным сегментам (бренды, категории, менеджеры)
- Динамика показателей по дням и неделям

### 3. Анализ эффективности
- **Оценка работы коммерческой команды:** сравнение результатов менеджеров, выявление лучших практик
- **Анализ товарного портфеля:** определение наиболее и наименее востребованных позиций
- **Сегментация брендов:** ранжирование по вкладу в выручку и объёму продаж
- **Выявление операционных проблем:** анализ причин отмен заказов и сбоев в процессах

### 4. Визуализация и отчётность
- Построение дашборда для мониторинга ключевых показателей
