# Wholesale Audio Equipment Sales Analysis

## Project Overview
Comprehensive analysis of wholesale audio equipment sales data including:
- Sales trend analysis and anomaly detection
- Brand performance and inventory optimization
- Sales manager performance evaluation

## Tech Stack
- Python 3
- Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook

## Key Insights
- 47.7% of orders on March 14 were reorders due to a system failure
- 30% of inventory items (497 SKUs) were never ordered
- Top manager contributed 17.04% of total revenue
